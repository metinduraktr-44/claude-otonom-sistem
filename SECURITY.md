# Security Policy — claude-otonom-sistem

## Reporting a vulnerability
Please report security issues privately via GitHub Security Advisories
(Security → Advisories → Report a vulnerability) or email **metindurak.tr@gmail.com**.
Response target: **72 hours**. Do not open public issues for exploitable findings.

**TR:** Güvenlik bulgusunu private advisory ya da e-posta ile bildir; kamuya açık issue açma. Yanıt hedefi 72 saat.

## Scope
claude-otonom-sistem (sistem/otomasyon). Supported: default branch (`main`).
